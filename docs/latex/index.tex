This example shows how to configure the UART module as loopback mode. The example receives a character from the UART trasnmitter and echoes it back to the same UART transmitter. The example uses the following peripherals and I/O signals. You must review these and change as needed for your own board\+:
\begin{DoxyEnumerate}
\item UART peripheral
\item GPIO Port peripheral (for UART pins)
\item UART\+\_\+\+TXD
\item UART\+\_\+\+RXD
